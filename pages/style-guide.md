## Colour Palette

Every colour is grounded in the island.

| Colour | Hex | Meaning | Use |
|--------|-----|---------|-----|
| **Teal** | `#0D7377` | The sea, the harbours | Primary brand, navigation, links |
| **Teal Dark** | `#065A5C` | Deep water | Hover states, emphasis |
| **Teal Light** | `#E0F2F2` | Shallow water | Backgrounds, highlights |
| **Deep Red** | `#8B1A1A` | The flag, the three legs | The book sections, Manx identity |
| **Gold** | `#C5A030` | Royalty, the Stanleys, the Atholls | Accents, highlights, nobility |
| **Slate Grey** | `#5A6672` | Sea mist, the stone | Endnotes, sources — scholarly sections |
| **Deep Blue** | `#1A3A5C` | The Irish Sea in deep water | Gallery |
| **Deep Green** | `#2D5A3D` | The land, the hills | Teaching section |
| **White** | `#FAFAFA` | The sky, clarity | Backgrounds |
| **Warm White** | `#FAF8F5` | Parchment | Text backgrounds |
| **Near Black** | `#1C1C1C` | Ink | Headings |
| **Text Grey** | `#333333` | — | Body text |

## Typography

**Headings:** Georgia (serif) — traditional, readable, appropriate for historical content

**Body text:** Source Sans Pro (sans-serif) — clean, modern, highly legible on screen

**Monospace:** Source Code Pro — for any code or technical content

### Sizes

- H1: 2.4rem
- H2: 1.8rem
- H3: 1.4rem
- H4: 1.2rem
- Body: 1rem (18px base)
- Small: 0.9rem

## File Naming

- Pages: lowercase with hyphens (`the-running-trade.md`)
- Images: lowercase with hyphens (`tynwald-hill-ceremony.webp`)
- All images in webp format for web delivery
- Original high-res files kept separately for print use
